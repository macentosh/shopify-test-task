# Shopify Test Task — Featured Products Section
Test task for a Refactor Plus

Цей репозиторій містить реалізацію тестового завдання для Shopify:
кастомна секція **Featured Products**, яка:

- виводить список рекомендованих товарів з обраної колекції;
- не показує товари, які вже є в кошику;
- додає товар до кошика через **AJAX** (без перезавантаження сторінки);
- оновлює mini-cart (cart drawer) та bubble-лічильник в хедері;
- написана з використанням **BEM-неймінгу**.

---

## 1. Вимоги

Для роботи з проєктом потрібні:

- **Node.js**
- **npm**
- **Shopify CLI**
- Доступ до **Shopify store** (development / test-store) з правами для роботи через Shopify CLI
- **Git** (опціонально, для роботи з репозиторієм)
